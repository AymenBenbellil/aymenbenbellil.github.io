What Operating System do I use?

I use Linux as my main OS, and precisely Arch Linux, even though I occasionally use a debloated Windows 10 too.

    I chose Linux instead of Windows because:
  • Linux Has all the tools I need to get work done (like: gcc, vim, mpv, yt-dlp, librewolf, zathura...).
  • Linux Has No Spyware or telemetry.
  • Linux Is Easier to use than Windows.
  • Linux Is Open Source and Free as in Freedom.
  • Linux Has No bloat.

Why did I choose Arch Linux?

Because it lets me control a big part of my OS unlike other linux distros. Plus, it's easy to use and learn.

Why I still use Windows & how did I debloat it?

I use Windows because of school, I have a module in which I must use both Microsoft Word and Exel.
I debloated it using the WinUtil that Chris Titus made.
